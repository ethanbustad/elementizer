Spell words using chemical element symbols.

Examples:
```
$ ./elementize.py bar
B   Ar
5   18
--------
B   A
o   r
r   g
o   o
n   n
```
```
$ ./elementize.py Liferay
Li  Fe  Ra  Y
3   26  88  39
----------------
L   I   R   Y
i   r   a   t
t   o   d   t
h   n   i   r
i       u   i
u       m   u
m           m
```
